## 项目概览

ParkingBuddy 是一款基于微信小程序的停车月卡在线销售平台。通过整合物业、开发商等甲方的空闲月卡资源，以批发价采购后面向个人车主零售，赚取差价和返点。

## 快速开始

### 1. 环境准备

- [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 最新版
- Node.js 16+
- npm

### 2. 修改配置

**`project.config.json`**：
```json
{
  "appid": "你的小程序AppID"
}
```

**`miniprogram/app.js`**：
```javascript
wx.cloud.init({
  env: '你的云环境ID',  // 替换为实际云环境
  traceUser: true,
});
```

### 3. 安装依赖 + 构建

```bash
# 进入小程序目录
cd parkingbuddy-miniapp/miniprogram

# 安装 TDesign
npm install

# 在微信开发者工具中：
# 工具 -> 构建 npm
```

### 4. 部署云函数

在微信开发者工具中，右键 `cloudfunctions/` 下每个云函数目录，选择"上传并部署"。

### 5. 创建数据库集合

在云开发控制台 -> 数据库中手动创建以下集合：

| 集合名 | 说明 | 权限 |
|-------|------|------|
| `parking_lots` | 停车场项目 | 所有用户可读，仅管理员可写 |
| `packages` | 月卡套餐 | 所有用户可读，仅管理员可写 |
| `orders` | 订单 | 仅创建者可读写 |
| `users` | 用户 | 仅创建者可读写 |
| `coupons` | 优惠券模板 | 所有用户可读，仅管理员可写 |
| `user_coupons` | 用户优惠券 | 仅创建者可读写 |
| `favorites` | 收藏 | 仅创建者可读写 |
| `reviews` | 评价 | 所有用户可读，仅创建者可写 |

### 6. 导入停车场数据

参考 `scripts/import_from_excel.js` 脚本导入 Excel 数据。

## 数据导入指南

### Excel 表格式要求

你的 Excel 表需要包含以下字段（一行一条记录）：

| 列名 | 示例 | 必填 |
|------|------|------|
| 项目名称 | 科技园南区停车场 | ✅ |
| 项目地址 | 深圳市南山区科技园南路88号 | ✅ |
| 收费标准 | 首小时5元，之后2元/小时，日封顶30元 | ❌ |
| 套餐名称 | 日间月卡（工作日） | ✅ |
| 套餐时限 | 工作日 8:00-20:00 | ✅ |
| 套餐原价 | 600 | ✅ |
| 套餐现价 | 480 | ✅ |
| 套餐推荐 | 是 | ❌ |
| 项目图片 | https://xxx.com/pic.jpg | ❌ |
| 项目标签 | 地下停车场,充电桩 | ❌ |
| 经度 | 113.9526 | ❌ |
| 纬度 | 22.5417 | ❌ |

### 导入步骤

1. 在云开发控制台 -> 数据库 -> `parking_lots` 集合 -> 导入 -> 选择 JSON/CSV
2. 或者通过云函数脚本导入（推荐）
