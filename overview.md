# 06-05 更新 — 五大需求交付

## 1. 微信实名认证登录
- `mine` 页登录改为 `open-type="getPhoneNumber"` 一键获取手机号
- 手机号 = 实名认证（中国手机号必须身份证实名才能注册）
- `login` 云函数新增 `phoneLogin` action：code2Session获取sessionKey → AES解密手机号 → 脱敏存储
- 登录后显示 "✓ 已实名" + 脱敏手机号

## 2. 常用地址多地址管理
- "出行地址" 改名 "常用地址"
- 三个预设标签：🏢 公司 / 🏠 家 / 📍 其他
- "其他" 可添加多项自定义地址，支持删除
- 数据存储: `wx.setStorageSync('addressInfo', { company, home, otherAddresses })`

## 3. 订单多选删除
- 长按订单进入编辑模式（震动反馈）
- 支持全选、单选、批量删除
- `order` 云函数新增 `delete` action
- 底部提示 "长按订单可进入多选编辑模式"

## 4. 优惠券分享
回答：是的，`onShareAppMessage` 在开发者工具中无法拉起微信分享面板。需要：
- 真机预览/体验版 → 点击右上角 "···" → 发送给朋友
- 或在预览二维码扫码后自然触发分享

## 5. 车辆表单全部选择器化
- **车牌号**：31省简称选择弹窗 + 城市字母输入 + 数字输入 → 自动拼接
- **品牌车型**：40+常见品牌选择器弹窗
- **车辆颜色**：12色预设色板（圆形色块点选）
